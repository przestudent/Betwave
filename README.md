# BetWave

Place bets on sport matches and see them in your dashboard!

It's a simple proof of concept site for a imaginary betting site. Made as a final project for one of my college classes.

-www.betwavepkb.live/

## Technologies used

- Next.js
- TypeScript
- Prisma
- PlanetScale

## Installation
```
npm install 
```
```
npm run dev 
```
